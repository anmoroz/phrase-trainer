## Simple application-trainer new English phrases and words

MySQL

Load mysql dump ./docker/mysql/phrase-trainer.sql

### Installation
```
 npm install
```

### Build
```
npm run build
```

### Start http server
```
npm start
```

Open http://localhost:8025

### Use docker
```
docker-compose up -d --build
```
